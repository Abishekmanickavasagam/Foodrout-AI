# FoodRoute AI – Smart Food Redistribution System

## Overview
FoodRoute AI is an intelligent food redistribution platform designed to reduce food waste and improve food accessibility. The system uses Deep Reinforcement Learning (DQN) and Multi-Agent Systems to optimize surplus food delivery from restaurants to NGOs and homeless individuals.

## Features
- Smart food redistribution
- Deep Reinforcement Learning (DQN)
- Multi-Agent coordination
- Route optimization
- Real-time logistics monitoring
- Google Maps API integration

## Tech Stack
- Python
- PyTorch
- Deep Reinforcement Learning
- Multi-Agent Systems
- Google Maps API
- Pandas
- NumPy


## Results
Improved food redistribution efficiency through intelligent routing and optimization.
